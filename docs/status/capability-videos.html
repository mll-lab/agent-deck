agent-deck Capability Videos

Watchable proof of the capability E2E flows

Each clip records the real agent-deck binary in an isolated sandbox (scratch HOME plus a per recording tmux socket via [tmux].socket_name, so no live session is ever touched). Nothing on screen is scripted text: the echobot reply you see is the genuine agent output. The headline clip drives the actual interactive TUI — you watch the session list, the selection moving, a live attach into a session pane, and the agent echoing a token back on screen — not commands typed at a prompt. VHS only supplies the paced terminal and the WebM encode. These are an on demand artifact, not part of the release gate. The pass or fail signal lives in capability-dashboard.html; these videos let you watch the same flows run.

Headline · Interactive TUI

Drive the real TUI: navigate, attach, watch the agent reply

Open the interactive agent-deck TUI, move the selection onto a live echobot session, press Enter to attach into its real tmux pane, type the token PING-TUI-9f3a, and watch the agent echo it back as ECHO:PING-TUI-9f3a live — then Ctrl+Q to detach and q to quit. This is the interface working, not CLI text being written.

Keybindings confirmed from internal/ui/home.go + internal/ui/hotkeys.go. The echo is genuine agent output captured live in the attached pane. Asserting twin: TestCapability_Agent_EchoRoundTrip.
